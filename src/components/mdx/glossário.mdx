## glossario-seven.vercel.app

[glossario-seven.vercel.app](https://glossario-seven.vercel.app) 
é uma aplicação para a web escrito em _typescript_, 
uma linguagem de programação. ele foi estruturado com [nextjs](https://nextjs.org/),
e hospedado na [Vercel](https://vercel.com), que usa o github como 
repositório. neste projeto, as palavras digitadas na caixa de texto
são pesquisadas numa base de dados que contém entradas oriundas
de diversos dicionários, como dicionário de sinômimos, dicionário de dificuldades,
dicionário de locuções e expressões e dicionário analógico. Apenas a última
palavra digitada é pesquisada, possibilitando uma escrita mais dinâmica.
