## microenem.vercel.app

[microenem.vercel.app](https://microenem.vercel.app) 
é uma aplicação para a web escrita em _typescript_,
uma linguagem de programação. Sua função é, grosso modo, apresentar 
informações estatísticas oriundas dos Microdados do ENEM, de 2019 até hoje.
Ela foi estruturada com [nextjs](https://nextjs.org/) e hospedada na
[Vercel](https://vercel.com), que usa o Github como 
repositório — tudo grátis. O código do aplicativo é fechado,
mas o da análise exploratória, feito com linguagem R, é aberto e pode ser
conferido [aqui](https://github.com/renribsilva/microEnemAnalyze). Na forma 
de _dashboard_, o aplicativo apresenta conteúdos que incluem, dentre outros,
o número de inscrições no exame, a reprodução da nota do Enem
e o impacto virtual de cada questão sobre a nota final.
