## microenem.vercel.app

[microenem.vercel.app](https://microenem.vercel.app) 
é um dashboard, cuja função é visualizar informações estatítiscas oriundas dos 
Microdados de Enem, a partir do ano de 2019. Escrito em _typescript_, 
uma linguagem de programação, ele foi estruturado com [nextjs](https://nextjs.org/) e hospedado na
[Vercel](https://vercel.com), que usa o Github como 
repositório — tudo grátis. O código do aplicativo é fechado, mas o da análise exploratória,
que foi feito com linguagem R e que gerou as informações apresentadas no saite, é aberto e pode ser
conferido [aqui](https://github.com/renribsilva/microEnemAnalyze). Tal conteúdo inclui
dados que vão desde o número de inscrições no exame até a reprodução da nota do Enem,
incluindo o impacto virtual de cada questão sobre a nota final.
