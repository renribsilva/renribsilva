## microenem.vercel.app

[microenem.vercel.app](https://microenem.vercel.app) 
é uma aplicação para a web voltada à apresentação de informações estatísticas
obtidas a partir dos Microdados do ENEM, de 2019 até hoje. 
Escrito em _typescript_, 
uma linguagem de programação, ele foi estruturado com [nextjs](https://nextjs.org/) e hospedado na
[Vercel](https://vercel.com), que usa o Github como 
repositório — tudo grátis. O código do aplicativo é fechado, mas o da análise exploratória,
feita com linguagem R, é aberto e pode ser
conferido [aqui](https://github.com/renribsilva/microEnemAnalyze). Na forma 
de _dashboard_, o aplicativo apresenta conteúdos que incluem, dentre outros,
o número de inscrições no exame, a reprodução da nota do Enem e 
o impacto virtual de cada questão sobre a nota final.
