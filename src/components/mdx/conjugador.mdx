## conjugador-gules.vercel.app

[conjugador-gules.vercel.app](https://conjugador-gules.vercel.app) 
é um _progressive web application_ (app com suporte para funcionar offline), cuja função é 
conjugar verbos da língua portuguesa brasileira. Escrito em _typescript_, 
uma linguagem de programação, ele foi estruturado com [nextjs](https://nextjs.org/) e hospedado na
[Vercel](https://vercel.com), que usa o Github como 
repositório — tudo grátis. neste projeto, cujo código é [aberto](https://github.com/renribsilva/conjugador), 
a base de vocábulos é a lista de palavras do corretor 
ortográfico do LibreOffice que pode ser acessada
livremente [aqui](https://github.com/LibreOffice/dictionaries/blob/master/pt_BR/pt_BR.dic)
— no projeto, ela foi salva em `pt-BR.dic` na pasta `/libreOfficeSource`.
Essa base de vocábulos contém mais de 312 mil palavras, dentre as quais
cerca de 19 mil são verbos terminados em 'ar', 'er', 'ir' e 'por'.
