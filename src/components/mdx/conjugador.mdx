## conjugador-gules.vercel.app

[conjugador-gules.vercel.app](https://conjugador-gules.vercel.app) 
é um conjugador de verbos da língua portuguesa para a web escrito em _typescript_, 
uma linguagem de programação. os objetivos deste projeto foram conhecer novas palavras no processo 
de desenvolvimento ter alguma diversão (por que não?).
ele foi estruturado com [nextjs](https://nextjs.org/) 
e [postgresql](https://www.postgresql.org/), dois recursos totalmente
gratuitos dentro da hospedagem da [Vercel](https://vercel.com), que usa o github como 
repositório, também grátis. neste projeto, a base de vocábulos é a lista de palavras do corretor 
ortográfico do libreoffice, que pode ser acessada
livremente [aqui](https://github.com/LibreOffice/dictionaries/blob/master/pt_BR/pt_BR.dic)
— no projeto, ela foi salva em `pt-BR.dic` na pasta `/libreOfficeSource`.
Essa base de vocábulos contém mais de 312 mil palavras, dentre as quais
cerca de 19 mil são verbos terminados em 'ar', 'er', 'ir' e 'por'.
